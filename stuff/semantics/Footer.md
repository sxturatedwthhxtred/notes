### Description
- `position: fixed;` -- attaches the footer to the bottom of page
- `width: 100%;` -- covers the entire width

%% its completely useless but not very bad experience i guess %%

```css
.footer {
  position: fixed;
  bottom: 0;
  width: 100%;
  background: black;
  color: white;
  text-align: center;
  padding: 10px;
}
```
